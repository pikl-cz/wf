## Implementation

```bash
# Install via Yarn or NPM
$ yarn add wifr

# CSS
node_modules/wifr/dist/wifr.min.css

# SCSS
@import '/node_modules/wifr/src/scss/index.scss';

# Additional JS
import '/node_modules/wifr/src/scss/index.scss';
```

## Build Setup

```bash

# install dependencies
$ yarn install

# serve with hot reload at localhost:9000
$ yarn start

# build for production
$ yarn build
```
